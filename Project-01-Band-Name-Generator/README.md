# 🎸 Day 01 - Band Name Generator

## 📌 Project Description

This is a simple Python project that generates a fun band name based on user input.
The user provides the name of the city they grew up in and their pet’s name, and the program combines them to suggest a band name.

---

## 🚀 Features

* Takes user input
* Combines strings to generate output
* Beginner-friendly logic
* Interactive command-line program

---

## 🧠 What I Learned

* Python `print()` and `input()`
* String concatenation
* Writing interactive programs
* Basic project structuring

---

## 💡 Example Output

```
Welcome to Band Name Generator!
In which have you grown up?
Rajmahal
What is your pet name?
kajaj
Your band name could be Rajmahal kajaj.
```
